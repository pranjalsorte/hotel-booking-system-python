# Simple Hotel Booking System (Python)

##  Project Overview
This is a menu-driven **Hotel Booking System** developed using **Python** and **file handling**.  
The application manages customers, room bookings, billing, and checkout without using a database.

##  Features
- Add and manage customer records
- View available hotel rooms
- Book rooms and calculate total bill
- View all bookings
- Checkout and update room availability
- Persistent storage using text files

##  Technologies Used
- Python 3
- File Handling
- VS Code

##  How to Run the Project
1. Clone the repository or download the project
2. Open the folder in VS Code
3. Run the program using:
   ```bash
   python hotel_booking.py
